# 目录介绍
## sample
### 样例目录
内有输入和输出样例，供测试用

你可以更改输入样例，只要符合题目要求即可，而后在 `src` 目录下运行 `./gen_output.sh` 以重新生成输出样例。

## src
### 源代码目录
输出样例正是由该目录下的程序生成。

## practice
### 练习目录
在这里添加你自已的代码，使用 `./test.sh` 来验证你的代码。

开始你的练习吧！

Tips:
+ 为了让 `test.sh` 正常工作，你在 `practice` 目录下文件的命名应该与 `src` 目录下的一致，但后缀名不一定要一致。
